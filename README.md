# Tetris_v1.0
## How to use it?
-Install Python 2.7 and Pygame  
-Open Tetris.py  
or
-Open Tetris.exe
## Operation
↓ down  
← left  
→ right  
Z anticlockwise rotation  
X clockwise rotation  
R restart  
P pause
